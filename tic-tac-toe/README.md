# TIC-TAC-TWO

# To play the game:
* Run ConsoleApp or
* Run WebApp

Your account is shared between console and web app.


# To change the saving mode from JSON to DB (or vice versa):
* Change "Mode" at Common/Settings.cs - Options are ESavingMode.Json and ESavingMode.Database.
* Other game settings and constants can also be changed at Settings.cs.

Keep in mind, that JSON and DB don't share the same games, configrations etc.
